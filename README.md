# Where's Waldo?

## Features:
1. Leaderboards with time
2. 3 Characters and 1 game map
3. Backend validation for character coordinates

## Steps:
1. Run `pnpm i` on both frontend and backend
2. Fill in the .env.template and move (rename) it to .env
3. Run `npx prisma generate && npx prisma migrate dev` on the backend
4. Run `npm run dev` on both the backend and frontend
5. Profit??

### Enjoy!
